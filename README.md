## About

This is a static personal website built with vanilla HTML and CSS. It serves as a professional hub for my career, projects, and technical background in cloud security and solutions engineering.

## Stack

- HTML5
- CSS3 (custom properties, CSS Grid, Flexbox)
- No frameworks or dependencies
- Hosted on GitHub Pages
- Custom domain via zodops.io

- **About** — Background, core focus areas, and in-progress certifications (AWS SAA, CKA)
- **Projects** — Open-source scripts and tooling (Go, Python, Bash)
- **Lab Notes** - Previous and current projects in the respective technology
- **Contact** — LinkedIn connection
